# 搜索引擎提交指南 - Stratronix网站

## 目标
让网站在Google、百度等全球主要搜索引擎中被收录和索引，特别是关于"AI AGENT"的搜索要能搜到我们的网站。

## 立即执行步骤

### 1. Google Search Console (最重要)
**网址**: https://search.google.com/search-console

**步骤**:
1. 使用Google账号登录
2. 点击"添加资源"
3. 选择"网址前缀"方式，输入: `https://www.stratronix.ai/`
4. 验证所有权（推荐使用HTML标签方式）:
   - 复制提供的meta标签
   - 添加到网站首页的`<head>`部分
   - 回到Search Console点击验证
5. 验证成功后，提交sitemap:
   - 左侧菜单点击"Sitemaps"
   - 输入: `https://www.stratronix.ai/sitemap.xml`
   - 点击"提交"
6. 请求索引:
   - 左侧菜单点击"网址检查"
   - 输入首页URL: `https://www.stratronix.ai/`
   - 点击"请求编入索引"

**关键设置**:
- 设置目标国家/地区
- 检查移动设备适合性
- 监控索引覆盖率

### 2. Bing Webmaster Tools
**网址**: https://www.bing.com/webmasters

**步骤**:
1. 使用Microsoft账号登录
2. 点击"添加网站"
3. 输入: `https://www.stratronix.ai/`
4. 验证所有权（XML文件方式最简单）
5. 提交sitemap: `https://www.stratronix.ai/sitemap.xml`
6. 提交URL进行抓取

### 3. 百度站长平台 (Baidu Webmaster Tools)
**网址**: https://ziyuan.baidu.com

**步骤**:
1. 使用百度账号登录（可能需要中国手机号）
2. 点击"添加网站"
3. 输入: `https://www.stratronix.ai/`
4. 验证所有权（推荐使用CNAME或文件验证）
5. 提交sitemap
6. 百度对中文内容更友好，确保中文页面优化

### 4. 其他重要搜索引擎

#### Yandex Webmaster (俄罗斯市场)
- 网址: https://webmaster.yandex.com
- 主要针对俄语用户

#### Naver Webmaster (韩国市场)
- 网址: https://searchadvisor.naver.com
- 需要韩语内容

#### DuckDuckGo
- 自动从Bing获取结果
- 确保Bing索引良好即可
