🎯 部署步骤

步骤1：访问 Vercel

打开浏览器，访问：https://vercel.com

步骤2：登录账号

使用以下方式登录：

  • GitHub 账号（推荐）
  • GitLab 账号
  • 电子邮件注册

步骤3：创建新项目

点击 "New Project" 按钮

步骤4：上传网站文件

将 stratronix.com-website 文件夹拖到上传区域

# 或者使用命令行
npm install -g vercel
cd stratronix.com-website
vercel --prod

步骤5：配置设置

  • Framework： 选择 "Static"
  • Build Command： （留空）
  • Output Directory： （留空）
  • 点击 "Deploy"

步骤6：获得临时地址

部署完成后获得：https://stratronix-xxx.vercel.app

测试访问确保网站正常显示

步骤7：添加自定义域名

在 Vercel 项目设置中：

  1. 点击 "Settings" → "Domains"
  2. 添加域名：stratronix.ai
  3. 添加域名：www.stratronix.ai
  4. 等待验证
