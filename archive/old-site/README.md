# Stratronix / 鼎图 双语品牌网站

## 🎯 项目概述
一个完整的双语企业网站，支持 Stratronix（英文品牌）和 鼎图（中文品牌），完全响应式设计，适合电脑、平板、手机查看。

## 🌐 网站结构
```
stratronix-website/
├── index.html              # 语言选择页面
├── en/                     # 英文版本
│   ├── index.html         # 英文首页
│   ├── about.html         # 关于我们（英文）
│   ├── products.html      # 产品介绍（英文）
│   ├── news.html          # 新闻动态（英文）
│   └── contact.html       # 联系我们（英文）
├── zh/                     # 中文版本
│   ├── index.html         # 中文首页
│   ├── about.html         # 关于鼎图（中文）
│   ├── products.html      # 产品介绍（中文）
│   ├── news.html          # 新闻动态（中文）
│   └── contact.html       # 联系我们（中文）
├── css/                    # 样式文件
│   ├── style.css          # 主样式
│   └── bilingual.css      # 双语样式
├── js/                     # 脚本文件
│   ├── main.js            # 主脚本
│   └── language-switcher.js # 语言切换
├── images/                 # 图片资源
├── admin/                  # 后台管理
│   ├── index.html         # 后台登录
│   └── dashboard.html     # 控制面板
├── deploy.sh              # 部署脚本
└── README.md              # 项目说明
```
